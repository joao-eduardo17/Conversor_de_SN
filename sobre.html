SOBRE O SITE:

O site inicialmente consistia em converter os números binários, octais e hexadecimais para decimais, até a ideia se expandir para uma conversão de todos para todos, e futuramente, se expandir para medidas de peso e distância.

A ideia da programação da lógica do site seria inicialmente em Python, utilizando o Flask para interligar as páginas, porém, com o JavaScript, o projeto se tornou muito mais desafiador e ao mesmo tempo simples, pois a dificuldade de inserir a lógica no site não existia mais, porém o código em Python precisaria ser "traduzido" para o JavaScript.
